- Newsletters publishing on schedule

⚠️ **Yellow Light** (Monitor closely):
- API quota > 80% used
- Minor delays in generation
- Open rate 20-25%

🚨 **Red Light** (Immediate action):
- API failures
- Newsletter generation failed
- Payment processing errors
- Open rate < 20%
